About the Project

Project Overview

AgriSmart is an intelligent Fertilizer Recommendation System developed to assist farmers and agronomists in making precise, data-driven decisions. By leveraging Machine Learning and modern web technologies, the system analyzes key environmental and soil parameters — including temperature, humidity, moisture, soil type, crop type, and essential nutrients (Nitrogen, Phosphorus, Potassium) — to recommend the most suitable fertilizer.

The goal is to optimize crop yield, minimize fertilizer overuse, reduce environmental impact, and promote sustainable agriculture practices. This project was built as a complete end-to-end demonstration for an internship, showcasing full-stack development, database management, and deployment of a machine learning model in a production-like web application.

Problem Statement

Traditional fertilizer application often relies on guesswork or generalized guidelines, leading to over-fertilization, soil degradation, increased costs, and environmental pollution (e.g., water contamination due to nutrient runoff). Farmers need a smart, accessible tool that provides personalized recommendations based on real-time field conditions.

Objectives

  • ✅ Deliver accurate fertilizer recommendations using a trained ML model
  • ✅ Build a responsive, user-friendly web interface for easy adoption by farmers
  • ✅ Implement persistent storage of all predictions using SQLite for future analysis
  • ✅ Promote sustainable farming by reducing chemical waste and improving yield efficiency
  • ✅ Demonstrate full-stack development skills: Backend (Flask), Frontend (HTML/CSS), ML Integration, and Database
  • ✅ Create a professional portfolio project suitable for internship evaluation and job applications

Methodology & How It Works

  1. Data Collection & Preprocessing: Used agricultural datasets containing soil and weather parameters.
  2. Model Training: Trained a classification model (Random Forest / similar) using scikit-learn to predict optimal fertilizer type.
  3. Web Development: Built with Flask framework for backend logic and routing.
  4. Frontend Design: Modern dark theme with neon green accents for a professional agricultural feel.
  5. Database Integration: SQLite stores every user prediction with full details for history tracking.
  6. Deployment: Local server with debug mode, easily deployable to platforms like Heroku or Render.

Key Features

  • 🔮 ML-Powered Engine — Real-time predictions based on 6+ input parameters
  • 💾 Complete Prediction History — View, track, and analyze past recommendations
  • 📱 Responsive UI/UX — Works perfectly on desktop, laptop, and mobile devices
  • 🌍 Sustainability Focus — Encourages precise fertilizer usage for eco-friendly farming
  • ⚡ Fast Performance — Lightweight Flask backend with quick response times
  • 🔒 Error Handling & Validation — Robust input validation and user feedback
